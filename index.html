    Perú Consultas¶


    Consultas de DNI y RUC sin uso de captcha, con cero dependencias.

    Instalar¶

    Via composer desde packagist.org

    composer require giansalex/peru-consult

    Disponibles¶

    • Ruc (SUNAT)
    • Dni (RENIEC)

    Requerimientos¶

    • Ruc necesita la extension dom.
    • Dni necesita las extensiones gd.

    API¶

    Puede utilizar el API REST peru-consult-api
